# Hero Quiz App
This is a Complete Hero Quiz App in FLUTTER using a lot of features such as

* Responsive apps

* Splash Screen Animation

* Multiple Screens

* Timer (10 sec By Default)

* Result Page

* Questions From JSON

And a lot more...
